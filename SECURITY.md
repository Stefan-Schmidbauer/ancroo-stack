# Security Policy

For the overall Ancroo security policy, roadmap, and Phase 1 limitations, see the [central security policy](https://github.com/Stefan-Schmidbauer/ancroo/blob/main/SECURITY.md).

For detailed security guidance specific to the stack (credentials, network exposure, firewall), see [docs/security.md](docs/security.md).

## Reporting a Vulnerability

Please report security vulnerabilities through [GitHub's private vulnerability reporting](https://github.com/Stefan-Schmidbauer/ancroo-stack/security/advisories/new).

Do not open a public issue for security vulnerabilities.
